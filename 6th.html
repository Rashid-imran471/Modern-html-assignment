Difference between HTML and HTML5


HTML HTML 5
1.It did not support audio and video without the use of flash player support. 1. It supports audio and video controls with the use of audio and video tags.
2. It uses cookies to store temporary data. 2. It uses SQL databases and application cache to store offline data.
3. Vector graphics is possible in HTML with the help of various technologies such as VML, Silver-light, Flash, etc. 3. Vector graphics is additionally an integral a part of HTML5 like SVG and canvas.
4. HTML,Body , and Head tags are mandatory while writing a HTML code. 4. These tags can be omitted while writing HTML code.
